# websocket-pressure-test

nodejs版的websocket压力测试工具


### 安装

```shell
npm i websocket-pressure-test
```
### 运行

```shell
./node_modules/.bin/webSocketPressureTest ws://127.0.0.1:9502 10000
//node_modules/.bin/webSocketPressureTest 地址 并发数
```
或

```shell
./node_modules/.bin/webSocketPressureTest ws://127.0.0.1:9502 10000
//node_modules/.bin/webSocketPressureTest 地址 并发数
```
